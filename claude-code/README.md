# claude-code

Claude Code plugins, skills, and agents built in or targeted at Israel.

## Plugins

| Plugin | Description | Repo |
|---|---|---|
| [Israel-Agent-Skills](https://github.com/danielrosehill/Israel-Agent-Skills-Plugin) | General Israel/Hebrew-specific agent skills (Hebrew translation, typography, emergency readiness, regional lookups). Shopping is out of scope. | `danielrosehill/Israel-Agent-Skills-Plugin` |
| [Claude-Israel-Shopping-Plugin](https://github.com/danielrosehill/Claude-Israel-Shopping-Plugin) | Shopping plugin for Israeli residents — domestic retailers plus common overseas B2C channels (AliExpress, Amazon, iHerb). | `danielrosehill/Claude-Israel-Shopping-Plugin` |
